HttpClient注解
  1. HttpParams须以链式调用来初始化参数，params.set('test', 'test')是会返回一个新的HttpParams对象的

  2. 使用POST方式时，若不指定Content-Type，则：

    (1) Content-Type=application/x-www-form-urlencoded;charset=UTF-8

    (2) 参数默认格式为FormData

  3. 使用POST方式时，若指定Content-Type值，除FormData格式之外：

    (1) Content-Type = 设定的值

    (2) 参数默认格式为RequestPayload
